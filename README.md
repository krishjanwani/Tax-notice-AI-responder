# ⚖️ CA AI Assistant: Tax Notice Responder

A professional AI-driven tool designed for Chartered Accountants to automate the drafting of responses to official Income Tax notices.

## 🚀 Features
* **PDF Notice Extraction**: Automatically reads and parses text from uploaded Tax Notice PDFs.
* **Gemini 2.5 Intelligence**: Uses Google's latest AI to draft legally structured responses.
* **Professional Export**: Generates a downloadable **Microsoft Word (.docx)** file for easy editing and final letterhead placement.
* **Secure Architecture**: Implements Streamlit Secrets for encrypted API key handling.

## 🛠️ Tech Stack
* **Frontend**: Streamlit
* **AI Engine**: Google Gemini 2.5 Flash
* **Libraries**: `python-docx` (Word generation), `PyMuPDF` (PDF parsing), `google-genai` (AI connection).

## 📥 How to Use
1. Upload a PDF of an Income Tax Notice.
2. Click **Generate Draft**.
3. Review the AI-generated response on the screen.
4. Click **Download Draft as Word** to save the file locally.
